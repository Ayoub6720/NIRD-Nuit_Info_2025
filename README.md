NIRD – Nuit de l'Info Project
================================

Projet réalisé dans le cadre de la Nuit de l'Info à l'ENSISA.

Description
-----------
Ce projet a été développé en équipe dans le cadre de la Nuit de l’Info.
Le site NIRD a pour objectif de sensibiliser les utilisateurs à l’importance du reconditionnement des appareils électroniques.
Il met en avant les enjeux environnementaux, économiques et sociaux liés à la réduction des déchets numériques et encourage une consommation plus responsable.

Technologies utilisées
----------------------
- HTML
- CSS
- JavaScript
